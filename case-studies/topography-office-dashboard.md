# Topography Office Dashboard — Case Study

A private business dashboard built for a topography office to manage daily operations, clients, projects, documents, invoices, users, and internal workflow.

> **Note:** The source code is private because this project contains business logic and client-specific workflow structure. This case study presents the product, features, and technical approach without exposing private code.

## Business Problem

Topography offices usually manage projects, documents, clients, payments, and team operations using scattered tools such as Excel files, folders, paper documents, and WhatsApp messages.

The goal of this dashboard is to centralize the workflow into one internal web application.
